Descripción del Proyecto

Este sitio web tiene como objetivo principal proporcionar información detallada sobre el sistema esquelético humano, sus componentes, funciones y las principales enfermedades que pueden afectarlo.

Objetivos Específicos

  • Describir la estructura del sistema esquelético
  • Explicar las principales funciones del esqueleto
  • Identificar los diferentes tipos de huesos
  • Analizar las enfermedades más comunes del sistema óseo
